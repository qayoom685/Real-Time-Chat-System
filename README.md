# Real-Time-Chat-System

Real-Time Chat System

A real-time messaging platform built for seamless communication between interns, teams, or collaborative groups.
This system supports instant one-to-one and group chat using WebSockets and stores all messages securely in MongoDB.

Features

✓ Real-Time Messaging using Socket.IO
✓ One-to-One & Group Chats
✓ User Presence Tracking (online/offline status)
✓ Message History Storage in MongoDB
✓ Secure and Scalable Architecture
✓ Automatic Reconnection Handling

Tech Stack

Node.js – Backend server

Express.js – REST API

Socket.IO – Real-time communication

MongoDB / Mongoose – Database for users and messages

Outcome

A fast, interactive, and fully scalable chat system that enables users to communicate instantly, making collaboration efficient and organized.

How to Run
-npm install
-npm run dev


Make sure MongoDB is running and your .env file contains:

PORT=5000
MONGO_URI=your_connection_string
